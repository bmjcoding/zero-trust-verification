# Attribution

The `architecture-reviewer` agent, the `/architecture` and `/diagnose-bug` commands, and the
`architecture-and-strictness.md` / `feedback-loop-diagnosis.md` references adapt concepts and
discipline from Matt Pocock's engineering skills:

- Source: https://github.com/mattpocock/skills (skills/engineering)
- License: MIT
- Skills drawn from: codebase-design, improve-codebase-architecture, domain-modeling, diagnosing-bugs

Concepts reused: the deep-module vocabulary (module / interface / depth / seam / adapter /
leverage / locality), the deletion test, "the interface is the test surface", "one adapter is a
hypothetical seam, two is a real one", and the feedback-loop-first bug-diagnosis discipline.

This plugin is an independent reinterpretation for the Claude Code plugin format, not a
redistribution of the original skill files.
